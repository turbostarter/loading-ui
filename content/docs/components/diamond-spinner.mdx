---
title: Diamond spinner
description: An 8-bit diamond loader with pixel-stepped opacity for retro-flavored waiting states.
---

`DiamondSpinner` uses eight square pixels arranged around a diamond path, each
fading in sequence. It reads especially well in game-inspired interfaces,
compact status surfaces, and loading states that want a little more character
than a standard ring.

## Customization

`DiamondSpinner` keeps the original pixel choreography intact, so the main
controls are its overall `size-*` footprint and the foreground color it
inherits via `currentColor`.
